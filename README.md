# Simple-password-checker-using-c-language
I've made a simple password checker using c programming language

## 📌 Overview

This project is a simple C-based application that evaluates the strength of a password based on its length. It is designed for beginners to understand basic string handling and conditional statements in C.

## ⚙️ How It Works

The program prompts the user to enter a password and then measures its length. Based on the number of characters, the password is categorized into one of the following:

* **Weak**: Less than 7 characters
* **Moderate**: Between 7 and 11 characters
* **Strong**: More than 11 characters

## ▶️ How to Run

### Step 1: Compile

Use a C compiler like GCC to compile the program.

### Step 2: Execute

Run the compiled executable file and enter a password when prompted.

## 💡 Example

* Input: `hello123`
* Output: Password is classified as **Moderate**

## ⚠️ Limitations

* The program only evaluates password strength based on length.
* It does not consider:

  * Use of special characters
  * Combination of uppercase and lowercase letters
  * Inclusion of numeric digits
